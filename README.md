#  ShoppyGlobe – E‑commerce Cart Application

A modern e‑commerce shopping cart web application built using **React**, **Vite**, **Redux Toolkit**, and **CSS**.
It simulates a complete shopping experience — from browsing products to checkout and order confirmation.

---

##  Getting Started

Follow the steps below to run this project on your local machine.

---

##  Prerequisites

Make sure you have the following installed:

* **Node.js** (v16 or above recommended)
* **npm** or **yarn**
* **Git**

Check versions:

```bash
node -v
npm -v
git -v
```

---

##  Clone the Repository

```bash
git clone https://github.com/chaitali9497/ecommerce-cart.git
cd ecommerce-cart
```

---

##  Install Dependencies

Using npm:

```bash
npm install
```


## Run the Application

Start the development server:

```bash
npm run dev
```

---

##  Open in Browser

Once the server starts, open your browser and visit:

```
http://localhost:5173
```

##  Tech Stack

* **React (Vite)**
* **Redux Toolkit** – Global state management
* **React Router DOM** – Routing
* **CSS** – Styling
* **Custom Hooks** – Data fetching & logic separation

##  Features

* Product listing & product detail page
* Real‑time search with highlighted results
* Add, remove, increase & decrease cart items
* Dynamic price & quantity calculation
* Checkout flow with order summary
* Responsive UI (mobile, tablet, desktop)
* Custom 404 page
